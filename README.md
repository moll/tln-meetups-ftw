# Tallinn Meetups for the Wall (or Win)!

Show Tallinn tech meetup logos and Twitter streams together.

## Building

Compile Haml to HTML with: `make`

Run watcher that compiles automatically: `make watch`

## And then...

Enjoy the profit!
